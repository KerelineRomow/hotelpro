🏨 Hotel Management System
Project Overview
This is a fully functional web-based platform built with Python and Django designed to streamline hotel operations. It allows staff to manage room availability, handle guest bookings, and maintain a structured database of services.
Key Features
• Room Management: Add, edit, and track the status of rooms (Available, Booked, Maintenance).
• Booking Engine: Logic for calculating stay costs and preventing double-bookings.
• Multilingual Support: Implemented i18n for seamless switching between English and local languages.
• User Authentication: Secure login system for administrators and staff.
• Responsive Design: Optimized for both desktop and mobile devices using CSS/Bootstrap.
Tech Stack
• Backend: Python 3.x, Django Framework.
• Database: SQL (PostgreSQL/SQLite).
• Frontend: HTML5, CSS3, JavaScript.
• Tools: Git, GitHub, Django i18n.
How to Run
1. Clone the repository.
2. Install dependencies: pip install -r requirements.txt.
3. Run migrations: python manage.py migrate.
4. Start server: python manage.py runserver.
